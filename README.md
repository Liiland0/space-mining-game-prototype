# Space Mining Game Prototype

A Unity 2D space exploration and mining prototype developed as a personal game development project.

The project focuses on building game systems and learning how to structure larger-scale gameplay mechanics using Unity and C#.

---

## Core Gameplay
Players explore space, mine resources from asteroids, complete quests, and upgrade their ship to progress.

---

## Features

### World & Visual Systems
- Infinite scrolling background with parallax effect to simulate depth
- Dynamic asteroid generation based on configurable input parameters
- Audio integration for gameplay feedback

---

### UI Systems
- Minimap displaying player position, nearest objective, and other game objects
- Interactive inventory system with hover-based item descriptions
- Clean UI overlays for quest feedback and progression

---

### Gameplay Systems
- Destructible environment using tilemaps and raycast-based interactions
- Modular quest system with objective tracking and reward distribution
- Ship upgrade system allowing progression through gameplay improvements

---

### Data-Driven Architecture
- ScriptableObjects used for:
  - Asteroid types
  - Item definitions
  - Quest data
- Enables easy expansion without modifying core gameplay code

---

## Technologies Used
- Unity Engine
- C#
- Unity Tilemaps
- ScriptableObjects system
- Raycasting systems
- VFX Graphs
